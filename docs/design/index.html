Design principles

  • Native GNOME feel. Looks like Files / Settings; no custom chrome that fights the desktop.
  • Settings out of the main view. The clipboard popup is for clipboard content; settings move to their own window.
  • Day-grouped rows. Pinned → Today → Yesterday → Earlier this week → Older. Easier to scan than a flat list.
  • Color-coded type icons. Text / image / link / code / snippet each get a tinted leading icon — at a glance.
  • Hover-only actions. Pin and delete buttons are invisible until you hover; reduces visual noise.
  • Catppuccin throughout. Both themes share the same tokens, only the palette changes.
